# LinkedIn Post — ApplyPilot Open Source Announcement

---

I just open-sourced ApplyPilot — a Chrome extension I built to solve a problem every job seeker knows too well:

Filling out the same name, email, phone, resume, cover letter, and 30+ other fields... on every single application... across different ATS platforms.

ApplyPilot detects form fields on job application pages and auto-fills everything in one click — across Greenhouse, Lever, Ashby, Workday, LinkedIn Easy Apply, SmartRecruiters, and 10+ more platforms.

Here's what makes it different:

→ Works without API keys — no subscriptions, no accounts, no backend
→ 30+ field types — text inputs, dropdowns, file uploads, searchable selects, EEO fields
→ AI cover letters (optional) — generates a tailored letter referencing the specific job & company
→ Never auto-submits — you always review and click Submit yourself
→ Zero data collection — everything stays in your browser. No analytics. No telemetry.

I built this because I was spending 15-20 minutes per application just filling in the same information. Now it takes under 2 minutes.

The extension uses Shadow DOM isolation, multi-strategy field detection with confidence scoring, and native event dispatching to work reliably across different React/Angular form implementations.

It's MIT licensed. No strings attached.

If you're job searching — or know someone who is — give it a try.
If you're a developer — PRs are welcome. There's plenty of room to add new ATS support, improve field detection, or add i18n.
